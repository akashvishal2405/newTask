Selectors
Element Selector p {} Style applied to all paragraph elements
Class Selector .example Style applied to all elements with class="example"
ID Selector #unique# Style applied to the element with id="uniqueID"
Attribute Selector input [type="text"] Style applied to input elements with type="text"
Descendant Selector div p {} Style applied to all paragraph elements that are descendant of a div
Child Selector div > p {} Style applied to all paragraph elements that are directly children of a div
Adjacent Sibiling Selector h2 + p {} Style applied to the paragraph immediately following an h2
Pseudo-classes a:hover {} Styles applied when the a element is hovered over
Pseudo-elements p::first-line {} Styles applied to the first line of every paragraph element
